## Wordly Dictionary SPA

An interactive dictionary app built with HTML, CSS, and JavaScript. Users can search for a word and see its definition, part of speech, example usage, pronunciation, and synonymsâ€”all without page reloads.

## Features
- Word search input
- API integration with Free Dictionary API
- Dynamic DOM updates
- Audio playback for pronunciation
- Synonyms and example usage

## How to Use
1. Open `index.html` in your browser.
2. Type a word and click "Search."
3. Results display below including definition and audio button (if available).

## Technologies
- HTML5
- CSS3
- JavaScript (Fetch API)
- [Free Dictionary API](https://dictionaryapi.dev/)
